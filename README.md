# Overview

As I continue developing my software engineering skills, I am creating simple programs to strengthen my understanding of programming fundamentals, code structure, and software development practices. Building small applications helps me gain confidence with syntax, program execution, and problem-solving techniques that will be useful in more advanced projects.

This software is a basic "Hello World" program written in Python. When executed, the program displays the message "Hello, World!" on the screen.

The purpose of creating this software is to verify that my development environment is correctly configured and to practice writing, running, and understanding a simple Python program. It serves as a foundation for learning more complex programming concepts and developing larger software applications in the future.
